Spam Detection using Naive Bayes theorem is a machine learning method used to classify messages as Spam or Ham (Not Spam) based on probability.the Naive Bayes algorithm calculates the probability that a message is spam based on the words present in the message. It uses a training dataset containing many messages labeled as spam and not spam. From this data, the model learns which words are commonly found in spam messages, such as “win”, “free”, “offer”, “lottery”, etc.When a new message arrives, the algorithm checks the words in that message and calculates the probability of the message being spam or not spam. The class with the highest probability becomes the prediction.
